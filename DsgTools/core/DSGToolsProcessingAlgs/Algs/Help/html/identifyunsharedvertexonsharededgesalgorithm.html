DsgTools versão 4.7.1

Descrição:

Identifica as feições que não possuem vértices compartilhado nas camadas selecionadas.

Parâmetros:

• Camadas do tipo linha → Serão verificadas linhas para identificar nós que não são compartilhados pelas feições na interseção, incluindo entre diferentes camadas selecionadas. • Camadas do tipo polígono → Serão verificados polígonos para identificar nós que não são compartilhados pelas feições na interseção, incluindo entre diferentes camadas selecionadas. • Raio de busca → Indica o raio para que seja considerado vértice nao compartilhado. Se a camada estiver no sistema de coordenada geográfica, a unidade de medida será em graus decimais. 0.00001 graus decimais ≅ 1 metro. Atenção: A unidade de medida do raio é baseada no sistema de coordenadas da camada vetorial de entrada.

Exemplo de uso:

Modelagem EDGV3.0 na escala de 1:25.000 (EPSG):4674

• Camadas do tipo linha → false • Camadas do tipo polígono → veg_floresta_a e hid_massa_dagua_a • Raio de busca → 0,01

Exemplo de Flag:

veg_floresta_a e hid_massa_dagua_a com vértices não compartilhados.
