Help versão 1.0

Descrição:

Identifica linhas que não são coincidentes com interseções de linhas com polígonos. Útil para verificar elementos de interseção, como pontes, que estão em algum lugar que não é interseção de camadas como massa d'água e vias.

Parâmetros:

• Line Layer → Camada vetorial de linha que pode intersectar os polígonos da camada de polígono. • Filter expression for line layer → Expressão para filtrar a camada de linhas. • Polygon Layer → Camada vetorial de polígono que pode intersectar as linhas da camada de linha. • Filter expression for polygon layer → Expressão para filtrar a camada de polígonos. • Intersection Lines Layer → Camada vetorial de linha representando elementos de interseção das camadas anteriores. • Filter expression for intersection layer → Expressão para filtrar a camada de elementos de interseção, útil para filtrar apenas os valores que representam a interseção desejada, como pontes numa camada de elementos viários quaisquer. Atenção: Não confundir com Identify Missing Lines on Polygon Line Intersections, que verifica o contrário, interseções de linhas com polígonos que não coincidem com outras linhas.

Exemplo de uso:

Modelagem EDGV 3.0 Topo 1.3 na escala de 1:25.000 (EPSG):4674

• Line Layer → infra_via_deslocamento_l • Polygon Layer → cobter_massa_dagua_a • Intersection Lines Layer → infra_elemento_viario_l • Filter expression for intersection layer → "tipo" in (201, 202, 203, 204, 501, 401, 402) and "modal_uso" in (4,97)
