DsgTools versão 4.7.1

Descrição:

Identifica os vértices dos polígonos que estão próximos à camada de referência.

Parâmetros:

• Input → Camada de geometria do tipo polígono a ser processada em busca de vértices próximos a camada de referência. • Reference Polygons → Camada do tipo polígono a ser referência. • Search Radius → Um ângulo entre 0 e 0,00001 onde 0.00001 graus decimais ≅ 1 metro. Atenção: Independentemente do sistema de coordenadas da camada de entrada, o parâmetro do raio de busca é fornecido em ângulos decimais.

Exemplo de uso:

Modelagem EDGV3.0 na escala de 1:25.000 (EPSG):4674

• Input → hid_massa_dagua_a • Reference Polygons → aux_moldura_a • Search Radius → 0,0001

Exemplo de Flag:

Vértice da camada hid_massa_dagua_a próximo à camada de referência.
