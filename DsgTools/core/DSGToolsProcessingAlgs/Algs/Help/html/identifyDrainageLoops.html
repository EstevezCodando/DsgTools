DsgTools versão 4.7.1

Descrição:

Identifica loops em uma camada vetorial, onde um loop é definido como um vetor que se conecta novamente em si mesmo após algum percurso.

Parâmetros:

• Input → Camada vetorial da primitiva geométrica do tipo linha a ser verificada. • Build local cache of the input layer → Permite que a camada de entrada seja armazenada localmente.

Exemplo de uso:

Modelagem EDGV3.0 na escala de 1:25.000 (EPSG):4674

• Input → hid_trecho_drenagem_l • Build local cache of the input layer → false

Exemplo de Flag:

hid_trecho_drenagem_l que se conecta novamente em si após algum percurso.
