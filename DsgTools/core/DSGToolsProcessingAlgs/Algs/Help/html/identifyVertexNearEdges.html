Help versão 1.0

Descrição:

Identifica vértices de um polígono ou linha que estejam próximos de uma aresta do próprio polígono ou linha, útil para detectar situações em que o polígono "afina" demais.

Parâmetros:

• Camada de entrada → Camada vetorial do tipo linha ou polígono. • Raio de busca → Distância máxima para que se considere o nó de uma feição muito próximo a uma aresta desta mesma feição. Se a camada estiver no sistema de coordenada geográfica , a unidade de medida será em graus decimais. 0.00001 graus decimais ≅ 1 metro. Atenção: A unidade de medida do raio é baseada no sistema de coordenadas da camada vetorial de entrada.

Exemplo de uso:

Modelagem EDGV3.0 na escala de 1:25.000 (EPSG):4674

• Camada de entrada → veg_floresta_a e veg_campo_a • Raio de busca → 0,001
