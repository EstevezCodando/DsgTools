Help versão 1.0

Descrição:

Identifica geometrias multipartes(agregações ou composições de várias partes geométricas menores) em uma camada vetorial.

Parâmetros:

• Input layer → Camada de entrada que contém as geometrias a serem verificadas.

Exemplo de uso:

Modelagem EDGV3.0 na escala de 1:25.000 (EPSG):4674

• Input layer → hid_trecho_drenagem_l

Exemplo de Flag:

Feição de hid_trecho_drenagem_l formada pela combinação de várias partes diferentes.
