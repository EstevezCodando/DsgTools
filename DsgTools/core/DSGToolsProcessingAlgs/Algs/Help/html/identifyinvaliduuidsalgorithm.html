Help versão 1.0
DsgTools versão 4.7.1

Descrição:

Identifica e conserta atributos das camadas que contém uuid inválido.

Parâmetros:

• Camada(s) de entrada → Camada a ser processada em busca de uuid inválido. • Nome do atributo → Nome do atributo a ser verificado. • Consertar? → Conserta atributo automaticamente da camada. • Compare only within same layer → Comparar atributo apenas dentro da mesma camada. Atenção: A rotina também identifica feições multi partes.

Exemplo de uso:

Modelagem EDGV3.0 na escala de 1:25.000 (EPSG):4674

• Camada(s) de entrada → aer_pista_ponto_pouso_p • Nome do atributo → id • Consertar? → false • Compare only within same layer → false
