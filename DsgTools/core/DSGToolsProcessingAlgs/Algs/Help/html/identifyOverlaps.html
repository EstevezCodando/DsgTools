Descrição:

Identifica feições de camadas do tipo polígono e linha que se sobrepõem a outras.

Parâmetros:

• Input layer → Camada a ser processada em busca de sobreposições entre suas feições. Atenção: Só identifica sobreposições com a mesma camada.

Exemplo de uso:

Modelagem EDGV3.0 na escala de 1:25.000 (EPSG):4674

• Input layer → lml_area_densamente_edificada_a

Exemplo de Flag:

Camada lml_area_densamente_edificada_a com sobreposição.
