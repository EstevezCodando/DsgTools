DsgTools versão 4.7.1

Descrição:

Identifica geometrias com alta densidade de vértices.

Parâmetros:

• Input layer → Camadas a serem processadas em busca de vértices próximos. • Search Radius → Distância utilizada para determinar a densidade dos vértices. Se a camada estiver no sistema de coordenada geográfica , a unidade de medida será em graus decimais. 0.00001 graus decimais ≅ 1 metro. Atenção: A unidade de medida do raio é baseada no sistema de coordenadas da camada vetorial de entrada.

Exemplo de uso:

Modelagem EDGV3.0 na escala de 1:25.000 (EPSG):4674

• Input layer → hid_trecho_drenagem_l • Search Radius → 0,00001

Exemplo de Flag:

Camada hid_trecho_drenagem_l com vértices próximos.
