DsgTools versão 4.7.1

Descrição:

Este algoritmo provê uma interface de modo que o usuário consiga verificar se uma camada respeita um conjunto de regras para o preenchimento correto de atributos ou conjunto de atributos.

Parâmetros:

• Descrição → Descrição da regra a ser aplicada ao atributo. • Camada e atributo → Camada referência sob a qual será aplicada a regra e seu atributo selecionado. • Expressão → Expressão no formato QgsExpression que será aplicada ao atributo em questão. • Tipo de erro → Tipo do erro a ser atribuído à feição no arquivo de flags. • Color → Cor que será aplicada ao atributo na tabela de atributos da camada em questão. Atenção: para identificar curvas de nível faltando, é importante que o usuário corrija quaisquer erros de pontas soltas primeiro.

Exemplo de uso:

Modelagem EDGV3.0 na escala de 1:25.000 (EPSG):4674

• Descrição → Preencher Atributo • Camada e atributo → hid_trecho_drenagem_l|tipotrechodrenagem • Expressão → "tipotrechodrenagem" not in(4,5) • Tipo de erro → Preencher atributo • Color → #f50000(vermelho)

Exemplo de Flag:

Atributo tipotrechodrenagem não preenchidos na camada de hid_trecho_drenagem_l.
