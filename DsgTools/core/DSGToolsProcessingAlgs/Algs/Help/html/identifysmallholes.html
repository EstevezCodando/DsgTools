Help versão 1.0

Descrição:

Identifica buracos (anéis interiores) nos polígonos, cuja área é menor que o tamanho máximo especificado pelo usuário.

Parâmetros:

• Camada(s) de entrada → Camadas do tipo polígono a ser identificado buracos. • Tolerância → Tamanho de área máximo do burraco especificado. 1000 m² ≅ 0,01 Atenção: A unidade de medida do raio é baseada no sistema de coordenadas da camada vetorial de entrada.

Exemplo de uso:

Modelagem EDGV3.0 na escala de 1:25.000 (EPSG):4674

• Camada(s) de entrada → veg_floresta_a • Tolerância → 0,01

Exemplo de Flag:

Área de 263 m².
