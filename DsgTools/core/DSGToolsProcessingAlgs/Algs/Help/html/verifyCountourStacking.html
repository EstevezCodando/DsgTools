Help versão 1.0

Descrição:

Identifica curvas de nível que estão sobrepostas.

Parâmetros:

• Selecionar → Camada de curva de nivel a ser processada. • Layer com curva de nível → Atributo da camada contendo os valores das cotas. • Atributo que infora a depressão → Atributo da camada contendo o tipo curva de nível. • Equidistância → Equidistância entre as curvas de nível em metros. • Process is running inside model → Um parâmetro booleano que indica se o algoritmo está sendo executado dentro de um model do QGIS.

Exemplo de uso:

Modelagem EDGV3.0 na escala de 1:25.000 (EPSG):4674

• Selecionar → rel_curva_nivel_l • Layer com curva de nível → cota • Atributo que infora a depressão → tipocurvanivel • Equidistância → 10 • Process is running inside model → false

Exemplo de Flag:

rel_curva_nivel_l estão sobrepostas.
