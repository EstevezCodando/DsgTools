Help versão 1.0

Descrição:

Identifica seguimentos de linhas em uma rede que possuem pontas soltas, linhas nao segmentadas e linhas desconectadas da rede.

Parâmetros:

• Input lines → Camadas do tipo linha a ser processada em busca de identificar os erros em comparação com os filtros. • input is a boundary layer → Cria os limites da camada de entrada, quando for usado os filtros opcionais e o parâmetro Geographic Boundary. • Search radius → Parâmetro onde se define a distância que os seguimentos de linhas estão de onde deveriam estar conectados. Se a camada estiver no sistema de coordenada geográfica a unidade de medida será em graus decimais onde 0.00001 graus decimais ≅ 1 metro. • Linestring Filter Layer → Camada opcional do tipo linha onde será filtrado em comparação com input layer. • Polygon Filter Layer → Camada opcional do tipo poligono onde será filtrado em comparação com input layer. • Ignore dangle on unsegmented lines → Parâmetro onde pode ser definido para ignorar linhas não segmentadas. • Geographic Boundary → Fronteira geográfica da região (moldura), que servirão como limitantes. Atenção: A unidade de medida do raio é baseada no sistema de coordenadas da camada vetorial de entrada.

Exemplo de uso:

Modelagem EDGV3.0 na escala de 1:25.000 (EPSG):4674

• Input Layer → hid_trecho_drenagem_l • input is a boundary layer → true • Search radius → 0,00100 • Linestring Filter Layer → hid_trecho_drenagem_l • Polygon Filter Layer → false • Ignore dangle on unsegmented lines → true • Geographic Boundary → aux_moldura_a
