DsgTools versão 4.7.1

Descrição:

Identifica pontos que possam ter problemas de fluxo de drenagem, como sentidos diferentes.

Parâmetros:

• Input → Camada vetorial da primitiva geométrica do tipo linha a ser processada.

Exemplo de uso:

Modelagem EDGV3.0 na escala de 1:25.000 (EPSG):4674

• Input → hid_trecho_drenagem_l

Exemplo de Flag:

hid_trecho_drenagem_l com sentidos diferentes.
