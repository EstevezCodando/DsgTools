Help versão 1.0

Descrição:

Identifica interseções de linhas que não são coincidentes com pontos de interseção. Útil para verificar elementos de interseção, como pontes, faltando na interseção de camadas como trecho de drenagens e vias.

Parâmetros:

• First Line Layer → Camada vetorial de linha que pode intersectar outras linhas da segunda camada de linha. • Filter expression for first line layer → Expressão para filtrar a primeira camada de linha. • Second Line Layer → Camada vetorial de linha que pode intersectar outras linhas da primeira camada de linha. • Filter expression for first line layer → Expressão para filtrar a segunda camada de linha. • Intersection Point Layer → Camada vetorial de ponto representando elementos de interseção das camadas de linhas. • Filter expression for intersection layer → Expressão para filtrar a camada de elementos de interseção, útil para filtrar apenas os valores que representam a interseção desejada, como pontes numa camada de elementos viários quaisquer. Atenção: Não confundir com Identify Missing Line Intersections on Points, que verifica o contrário, pontos que não coincidem com interseções de linhas.

Exemplo de uso:

Modelagem EDGV 3.0 Topo 1.3 na escala de 1:25.000 (EPSG):4674

• First Line Layer → infra_via_deslocamento_l • Second Line Layer → elemnat_trecho_drenagem_l • Filter expression for second line layer → "situacao_em_poligono" = 1 • Intersection Point Layer → infra_elemento_viario_p • Filter expression for intersection layer → "tipo" in (201, 202, 203, 204, 501, 401, 402) and "modal_uso"=4
