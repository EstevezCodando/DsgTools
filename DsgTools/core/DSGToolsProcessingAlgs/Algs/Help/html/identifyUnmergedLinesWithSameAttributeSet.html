Help versão 1.0

Descrição:

Identifica linhas que possuam os mesmos atributos em comum, permitindo que elas sejam mescladas em uma única linha.

Parâmetros:

• Input Layer → Camada a ser processada em busca de polígonos pequenos. • Fields to ignore → Parâmetro onde podem ser selecionados atributos a serem ignorados no processamento. • Ignore virtual fields → Ignora campos dinamicamente existentes na tabela de atributos. • Ignore primary key fields → Ignorar campos de chave primária. • Point Filter Layers → Filtra camadas do tipo ponto que estão conectados às feições selecionadas para evitar a identificação. • Line Filter Layers → Filtra camadas do tipo linha que estão conectados às feições selecionadas para evitar a identificação.

Exemplo de uso:

Modelagem EDGV3.0 na escala de 1:25.000 (EPSG):4674

• Input Layer → rod_trecho_rodoviario_l • Fields to ignore → atributos a ser ignorados • Ignore virtual fields → true • Ignore primary key fields → true • Point Filter Layers → false • Line Filter Layers → false

Exemplo de Flag:

rod_trecho_rodoviario_l com mesmos atributos em comum.
