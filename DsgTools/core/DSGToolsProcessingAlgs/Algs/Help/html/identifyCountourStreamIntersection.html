Help versão 1.0

Descrição:

Identifica os pontos onde o trecho de drenagem intersecta uma mesma curva de nível, subindo e descendo a cota.

Parâmetros:

• Layer de trecho drenagem → Camada vetorial de linhas representando uma rede de drenagem. • Layer de curva de nível → Camada vetorial de linhas representando curvas de nível. • Process is running inside model → Um parâmetro booleano que indica se o algoritmo está sendo executado dentro de um model do QGIS. Atenção: Essa ferramenta não gera interseção nas classes entre si.

Exemplo de uso:

Modelagem EDGV3.0 na escala de 1:25.000 (EPSG):4674

• Layer de trecho drenagem → hid_trecho_drenagem_l • Layer de curva de nível → rel_curva_nivel_l • Process is running inside model → False

Exemplo de Flag:

hid_trecho_drenagem_l subindo e descendo rel_curva_nivel_l.
