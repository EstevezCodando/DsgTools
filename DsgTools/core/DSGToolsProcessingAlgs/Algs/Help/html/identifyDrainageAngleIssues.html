Help versão 1.0

Descrição:

Identifica possíveis problemas nos ângulos de drenagem em pontos de confluência, sendo capaz de detectar pontos em que o ângulo de drenagem está incorreto.

Parâmetros:

• Input → Camada vetorial de trecho drenagem a ser processada.

Exemplo de uso:

Modelagem EDGV3.0 na escala de 1:25.000 (EPSG):4674

• Input → hid_trecho_drenagem_l

Exemplo de Flag:

hid_trecho_drenagem_l com confluência errada (90°).
