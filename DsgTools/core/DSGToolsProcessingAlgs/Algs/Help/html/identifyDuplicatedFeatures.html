Help versão 1.0

Descrição:

Identifica feições com um mesmo conjunto de atributos e geometrias iguais.

Parâmetros:

• Camadas de entrada → Camada a ser processada em busca de geometrias duplicadas. • Campos a ignorar → Campo opcional de atributos que devem ser ignorados pelo algoritmo quando for comparar as feições. • Ignorar campos virtuais → Indica se o algoritmo deve levar atributos virtuais gerados pelo usuário em consideração quando for comparar feições. • Ignorar campos de chaves primárias → Indica se o algoritmo deve levar atributos usados como chaves primárias (ex. ID) gerados pelo usuário em consideração quando for comparar feições. Atenção: Só é considerado o conjunto de atributos das feições iguais.

Exemplo de uso:

Modelagem EDGV3.0 na escala de 1:25.000 (EPSG):4674

• Camadas de entrada → cbge_trecho_arruamento_l • Campos a ignorar → id|nome|observacao • Ignorar campos virtuais → true • Ignorar campos de chaves primárias → true

Exemplo de Flag:

Camada cbge_trecho_arruamento_l com feições duplicadas.
